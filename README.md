# Speak App

Check out the site [here](http://dbc-speak.herokuapp.com/)!

## Description:

Speak -- Augmentative and Alternative Communication(AAC) application for non-verbal people.

## Tech:

- Ruby on Rails
- PostgreSQL
- Javascript / jQuery
- AJAX
- Bootstrap
- Google Maps API, Cloudinary, AWS Polly
